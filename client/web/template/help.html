FAQ

Какого формата данные необходимо вводить в поля?

  • Формат играницы параметра для каждого поля указаны под каждым полем мелким тесктом

    • Как должен выглядеть файл для отправки?

  • Файл должен быть формата .csv
  • Максимальный размер файла - 5 MB
  • Разделителем должен быть символ ";"
  • Названий столбцов быть не должно
  • Каждый ряд должен состоять из трех чисел: размер, содержание и стабильность соостветсвенно
  • формат данных такой же как при введении в поля

    • Пример содержания фалйа:

    8.5;100;730
    16.09;100;730
    23;99.8;730
