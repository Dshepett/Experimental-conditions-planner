# Программа планирования эксперимента для синтеза золей с определенными характеристиками

- Приложение было разработано для курсовой работы
- Вычисляет условия для синтеза золей с определенными характеристиками
- Для вычисления условий синтеза использовался ML-флгоритм RandomForest
- Сервер был разработан на Python, а клиент на Go
- Ссылка на рабочую версию прилжения: https://conditions-planner.herokuapp.com/
